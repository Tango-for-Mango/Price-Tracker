# Price-Tracker
System do asynchronicznego monitorowania i analizy historii cen produktów w serwisach e-commerce

Celem projektu jest stworzenie aplikacji webowej pozwalającej użytkownikom na śledzenie cen wybranych produktów w sklepach internetowych (wersja MVP: X-Kom oraz Helion/Morele). Zamiast ręcznie sprawdzać oferty, użytkownik dodaje link do interesującego go przedmiotu. System działający w tle regularnie pobiera aktualne dane cenowe, zapisuje je w bazie danych i wizualizuje historię zmian na wykresach, co pozwala na świadome podejmowanie decyzji zakupowych i unikanie fałszywych promocji.
